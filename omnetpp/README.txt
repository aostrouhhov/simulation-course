ЗАДАЧА 20
----
Ошибка в формулировки вопроса.

"Не менее 2%" - хотим, чтобы програмисты работали. Тогда:
"Меньше программистов"" - общее время занятости больше
"Больше программистов" - время меньше

Следовательно, хотим найти не минимальное число программистов, а максимальное!
Можно поставить вопрос по-другому, не трогать программистов и сделать "не более 2%". Так даже правильнее было бы.

СТРУКТУРА
----
* omnet.ini - описание конфигурации проекта
* repair.ned - NED файл, описывающий модули и сеть (как связаны подмодули) // грубо говоря, модуль - это класс, а подмодуль - это экземпляр класса
* repair.cc - реализация классов Masters (для модуля Masters) и Server (для модуля Server): инициализация, обработка сообщений, уничтожение.

Создай новый проект -> Скопируй эти 3 файла -> Запускай!

В итоге вывод должен направляться в файл:
* Run -> Run Configurations -> Main -> User interface "Cmdenv"
* Run -> Run Configurations -> Main -> снять галочку "Verbose"
* Run -> Run Configurations -> Main -> поставить галочку (а не квадратик!) на "Save stdout"
* Run -> Run Configurations -> Common -> вписать "Output file"
* Run -> Run Configurations -> Common -> поставить галочку "Append"

Но для отладки удобнее пользоваться визуалиизрованной симуляцией и выводом! 
Если отлаживаешься, то вместо std::cout использовать EV
